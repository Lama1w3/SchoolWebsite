Admission Eligibility

  • Class VI: Passed Class V from recognized school.
  • Class IX: Passed Class VIII from recognized board.
  • Class XI: Passed Class X (HPBOSE/CBSE or equivalent).
  • Arts / Science / Commerce: As per merit and subject availability.

Required Documents

  • Transfer Certificate (TC)
  • Character Certificate
  • Previous Class Marksheet
  • Birth Certificate
  • Aadhar Card (Student & Parent)
  • Passport Size Photographs
  • Caste Certificate (if applicable)

Admission Procedure

  1. Collect Admission Form from School Office.
  2. Submit duly filled form with required documents.
  3. Verification of documents by school authority.
  4. Fee submission after approval.
  5. Admission confirmation & entry in school register.

Age Criteria

Minimum age for Class VI should be 10+ years. Age relaxation as per Government norms.

Transfer Certificate Rules

Transfer Certificate will be issued only after clearance of all dues. Application for TC must be submitted in writing by parent/guardian.
